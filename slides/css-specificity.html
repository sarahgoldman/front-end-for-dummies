Specificity Rules

  • Rules with more specific selectors have a greater specificity.
    #p123 is more specific than .p123
  • ID selectors have a higher specificity than attribute selectors.
    #p123 is more specific than [id=p123]
  • Contextual selectors are more specific than a single element selector.
    p a is more specific than a
  • The embedded style sheet is closer to the element to be styled.
    Rules defined in a style tag are more specific than external CSS
  • The last rule defined overrides any previous, conflicting rules. 
    #content h1 {
	padding: 5px;
}

#content h1 {
	padding: 10px; /* this will be used */
}
  • A class selector beats any number of element selectors.
    .introduction beats html body div div h2 p
