# Syntax

```
// Selector
p { padding: 10px; }

// Class selector
.thumbnail { background: red; }

// ID selector
#main_nav { font-style: italic; }

// Contextual selector
p span { color: green; }

// Attribute selector
p[title] { font-weight: bold; }

// Pseudo-classes
a:visited { text-decoration: underline; }

// Pseudo-elements 
p:first-line { font-variant: small-caps; }
```
